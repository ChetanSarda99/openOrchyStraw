# OrchyStraw (Private)

Private development repo for OrchyStraw — multi-agent AI coding orchestration.

## Repo Structure

```
research/          — Competitive analysis, pattern analysis, benchmarking plans
docs/              — Architecture references, design decisions
benchmarks/        — SWE-bench scaffold, Ralph vs Orchystraw head-to-head
```

## Related Repos

- **[openOrchyStraw](https://github.com/ChetanSarda99/openOrchyStraw)** — Public open-source repo (MIT). The coordination scaffold: markdown prompts + bash. Stable, community-facing.
- **OrchyStraw (this repo)** — Private. Research, benchmarking, Pixel Agents integration, Agent Factory development, proprietary improvements before they go open-source.
