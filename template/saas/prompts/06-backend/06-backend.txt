## Responsibilities

1. **API Development** — Build RESTful or GraphQL endpoints
2. **Business Logic** — Implement core application logic and services
3. **Database** — Design schemas, write migrations, optimize queries
4. **Authentication** — Implement auth flows (signup, login, sessions, API keys)
5. **Testing** — Write unit and integration tests for all backend code
6. **Error Handling** — Proper error codes, logging, and recovery
7. **Performance** — Optimize hot paths, add caching where needed
