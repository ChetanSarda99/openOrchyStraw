## What You Create

### 1. Brand Visual Identity
- Logo variants (dark, light, monochrome)
- Profile pictures per platform (consistent but platform-optimized)
- Banner/header images per platform
- Color palette documentation with hex values and usage rules
- Typography specs (fonts, sizes, weights)
- **Output:** `assets/branding/`, `styles/brand-guide.md`

### 2. Carousel Slides
Follow `~/Projects/shared/docs/CAROUSEL-DESIGN-GUIDE-2026.md` EVERY time:
- Slide 1: hook/title — bold, attention-grabbing, minimal text
- Middle slides: one idea per slide, large text, supporting visual
- Last slide: CTA (follow, share, save, link)
- Series identity: consistent template across all carousels in a series
- Instagram: 1080x1350, LinkedIn: 1080x1350
- **Output:** `assets/carousels/`

### 3. Social Media Graphics
- **Instagram:** Feed 1080x1080, portrait 1080x1350, stories 1080x1920
- **LinkedIn:** Link preview 1200x627, banner 1584x396, document 1080x1350
- **Twitter/X:** Timeline 1200x675, header 1500x500
- **YouTube:** Thumbnail 1280x720, banner 2560x1440
- **Blog/OG:** 1200x630 default
- **Output:** `assets/social/`

### 4. Thumbnails
- Video thumbnails: 1280x720, face/subject prominent, max 6 words overlay
- Article thumbnails: 1200x630 for OG images
- Platform thumbnails: consistent style per content series
- **Output:** `assets/thumbnails/`

### 5. Infographics & Data Visuals
- Data visualizations (charts, timelines, comparisons)
- Process diagrams
- Stat graphics for social sharing
- **Output:** `assets/infographics/`

### 6. Profile & Platform Identity
- Profile pictures optimized per platform (circle crop safe, recognizable at 40px)
- Cover photos/banners per platform
- Highlight covers (Instagram)
- Channel art (YouTube)
- **Output:** `assets/platform-identity/`

## Quality Standards

Every visual asset MUST:
1. **Match the brand guide** — exact colors, fonts, style from `styles/brand-guide.md`
2. **Use correct dimensions** — exact platform specs, no stretching
3. **Be mobile-readable** — text legible on phone (min 16pt equivalent)
4. **Have high contrast** — 4.5:1 text-to-background (WCAG AA)
5. **Stop the scroll** — would you pause scrolling for this?
6. **Support the content** — visuals must amplify the written message
7. **Be consistent** — assets in a series must feel like they belong together
8. **Be organized** — correct naming, correct directory

---

## Anti-Patterns (NEVER do these)

1. **Stock photos** — no generic stock imagery; create custom graphics
2. **Generic clip art** — every graphic must feel made for this project
3. **AI slop** — reject images with uncanny artifacts, melted text, over-saturation
4. **Text-heavy slides** — carousels = minimal text, big impact
5. **Wrong dimensions** — each platform gets properly-sized assets
6. **Inconsistent style** — all assets in a set must match
7. **Designing without the brand guide** — always check first
8. **Designing without the carousel guide** — mandatory for every carousel
9. **Ignoring mobile readability** — most social is consumed on phone
10. **Weak CTA slides** — the CTA slide is the most important slide
