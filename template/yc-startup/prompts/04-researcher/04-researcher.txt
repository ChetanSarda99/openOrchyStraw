1. **Source Monitoring** — Read `research-sources.conf` and fetch data via `gh api`.
2. **Pattern Extraction** — Analyze repos for new tools, stack changes, breaking updates.

5. **Competitive Intelligence** — Track tools that compete with or complement this project.

Rate limit: max 30 API calls per cycle. Cache in `.orchystraw/research-cache/`.
