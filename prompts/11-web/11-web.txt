## What is OrchyStraw?

Multi-agent AI coding orchestration. Markdown prompts + bash script. No framework, no dependencies. Works with Claude Code, Codex, Gemini, Aider, Windsurf, Cursor — anything that takes a prompt.

**Tagline options:**
- "Run a team of AI coding agents on any codebase"
- "Get multiple AI agents working on the same codebase"
- "Multi-agent AI orchestration. No framework. No dependencies."

**Repo:** https://github.com/ChetanSarda99/openOrchyStraw

## Design Inspiration

### Primary: Conductor (conductor.build)
- Clean, minimal, dark background
- Bold hero text with one clear value prop
- 3-step "How it works" section (numbered, visual)
- FAQ with expandable accordion
- Social proof bar ("Trusted by builders at...")
- Single CTA: "Get Started" or "Star on GitHub"
- No cluttered nav — just logo + a few links
- Smooth, not flashy. Developer-focused.

### Secondary: Claude Code Docs (code.claude.com/docs)
- Tabbed content (Terminal / VS Code / Desktop / Web)
- Accordion sections for features
- Code blocks with syntax highlighting
- Clean sidebar navigation for docs
- Structured, scannable, no walls of text

### Design Principles (also guides Tauri app UI)
- **Dark mode first** — near-black background (#0a0a0a or similar)
- **Monospace accents** — JetBrains Mono or Fira Code for code/technical text
- **Minimal color palette** — white text, one accent color (OrchyStraw brand)
- **No gradients** — flat, clean surfaces
- **Generous whitespace** — let content breathe
- **Developer tone** — technical but approachable, no marketing fluff
- **Mobile responsive** — works on phone, but desktop is primary
