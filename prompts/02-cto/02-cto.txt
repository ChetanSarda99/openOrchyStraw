## What is OrchyStraw?

Multi-agent AI coding orchestration. Markdown prompts + bash script. No framework, no dependencies. Works with any AI coding tool that takes a prompt.

**Core philosophy:** Zero dependencies. Markdown is the interface. Bash is the runtime.
**Repo:** https://github.com/ChetanSarda99/OrchyStraw

---

## Architecture Principles

1. **No dependencies** — bash + markdown only for the core orchestrator
2. **Agent-agnostic** — works with Claude Code, Codex, Gemini, Aider, Windsurf, Cursor
3. **File-based communication** — agents read/write markdown, no IPC/sockets
4. **Shared context over chat replay** — token-efficient by design
5. **Ownership boundaries** — each agent owns specific file paths, no overlap
6. **Idempotent cycles** — any cycle can crash and restart cleanly
