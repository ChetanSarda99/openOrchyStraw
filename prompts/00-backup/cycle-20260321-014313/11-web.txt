## What is OrchyStraw?

Multi-agent AI coding orchestration. Markdown prompts + bash script. No framework, no dependencies. Works with Claude Code, Codex, Gemini, Aider, Windsurf, Cursor — anything that takes a prompt.

**Tagline options:**
- "Run a team of AI coding agents on any codebase"
- "Get multiple AI agents working on the same codebase"
- "Multi-agent AI orchestration. No framework. No dependencies."

**Repo:** https://github.com/ChetanSarda99/OrchyStraw-Pro
**Public repo:** https://github.com/ChetanSarda99/openOrchyStraw

## Design Inspiration

### Primary: Conductor (conductor.build)
- Clean, minimal, dark background
- Bold hero text with one clear value prop
- 3-step "How it works" section (numbered, visual)
- FAQ with expandable accordion
- Social proof bar ("Trusted by builders at...")
- Single CTA: "Get Started" or "Star on GitHub"
- No cluttered nav — just logo + a few links
- Smooth, not flashy. Developer-focused.

### Secondary: Claude Code Docs (code.claude.com/docs)
- Tabbed content (Terminal / VS Code / Desktop / Web)
- Accordion sections for features
- Code blocks with syntax highlighting
- Clean sidebar navigation for docs
- Structured, scannable, no walls of text

### Design Principles (also guides Tauri app UI)
- **Dark mode first** — near-black background (#0a0a0a or similar)
- **Monospace accents** — JetBrains Mono or Fira Code for code/technical text
- **Minimal color palette** — white text, one accent color (OrchyStraw brand)
- **No gradients** — flat, clean surfaces
- **Generous whitespace** — let content breathe
- **Developer tone** — technical but approachable, no marketing fluff
- **Mobile responsive** — works on phone, but desktop is primary

### Section 2: How It Works (3 steps, like Conductor)
1. **Configure your agents** — "Define roles in agents.conf. Backend, frontend, QA — whatever your project needs."
2. **Run the orchestrator** — "One command. Agents read their prompts, do their work, write to shared context."
3. **Review and ship** — "PM agent coordinates. QA reviews. You merge."

### Section 3: Features Grid (4-6 cards)
- Zero dependencies (bash + markdown)
- Agent-agnostic (works with any AI tool)
- File ownership boundaries (agents can't step on each other)
- Shared context (token-efficient communication)
- Auto-cycle mode (run N cycles unattended)
- Pixel Agents visualization (agents working in pixel art office)

### Section 4: Supported Tools
- Logo row: Claude Code, Codex, Gemini CLI, Aider, Windsurf, Cursor, ChatGPT
- "Works with anything that takes a prompt"

### Section 5: FAQ (accordion, like Conductor)
- "What is OrchyStraw?" — Multi-agent orchestration via markdown + bash
- "Do I need to install anything?" — No. Just bash and your AI tool of choice.
- "How much does it cost?" — Free and open source (MIT)
- "Which AI models work?" — Any. Claude, GPT, Gemini, local models, whatever.
- "How is this different from AutoGen/CrewAI?" — No Python runtime, no dependencies, works with real coding agents not chat agents.

### Section 6: Footer
- GitHub link, MIT license, "Built with OrchyStraw" tagline
