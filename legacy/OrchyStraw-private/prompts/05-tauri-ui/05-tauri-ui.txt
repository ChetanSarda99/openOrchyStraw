## What is OrchyStraw?

Multi-agent AI coding orchestration. The desktop app gives users a GUI to manage
agent teams, trigger cycles, view logs, and watch agents work via Pixel Agents.

**Repo:** https://github.com/ChetanSarda99/OrchyStraw

## Tech Stack

**READ FIRST:** `docs/references/TAURI-STACK.md` — locked stack decisions, DO NOT deviate.

- **Base template:** dannysmith/tauri-template (clone, don't scaffold from scratch)
- **Framework:** React 19 + TypeScript + Vite 7
- **Components:** shadcn/ui v4 (copy-paste, Tailwind-based, shared with landing page)
- **Styling:** Tailwind CSS v4 + shadcn design tokens
- **IPC:** Tauri `invoke()` — types auto-generated by tauri-specta from Rust commands
- **State (UI):** Zustand v5 (sidebar, theme, etc.)
- **State (data):** TanStack Query v5 (persistent data from Rust)
- **Icons:** Lucide React
- **Theme:** Dark mode default (#0a0a0a), Inter/Geist (UI), JetBrains Mono (code)
- **MCP:** shadcn MCP server in `.mcp.json` — use to browse/install components
- **Quality:** ESLint + Prettier + ast-grep + Vitest
- **Philosophy:** Fast, minimal, data-dense. No unnecessary animations.

### Three-Layer State Rule
1. `useState` — component-local only
2. Zustand — global UI state
3. TanStack Query — persistent/async data from Rust

### Template Built-ins (already done, don't rebuild)
- Command Palette (Cmd+K), collapsible sidebars, dark/light mode
- Preferences system, notifications, auto-updates, keyboard shortcuts
- Native menus, i18n, crash recovery, structured logging
- Platform-specific title bars (macOS/Windows/Linux)

## Design Inspiration

### Primary: Conductor (conductor.build)
- Clean, dark, developer-focused desktop UI
- At-a-glance agent status — who's working, what needs attention
- Minimal chrome, data-dense panels
- Smooth, not flashy

### Also reference: Claude Code Docs (code.claude.com/docs)
- Tabbed content, accordion sections
- Syntax-highlighted code blocks
- Clean sidebar navigation

### Design Principles
- **Dark mode first** — near-black background (#0a0a0a), light mode secondary
- **Data density** — show more info, fewer clicks
- **Monospace for code** — JetBrains Mono or Fira Code
- **Minimal color palette** — white text, one accent color (OrchyStraw brand)
- **No gradients** — flat, clean surfaces
- **Generous whitespace** — let content breathe
- **Status colors** — 🟢 running, 🟡 idle, 🔴 error, ⚪ not this cycle
- **No loading spinners** — use skeleton/placeholder states
