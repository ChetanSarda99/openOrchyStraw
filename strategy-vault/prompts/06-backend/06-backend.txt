## What is OrchyStraw?

Multi-agent AI coding orchestration. The core is a bash script that reads `agents.conf`,
spawns AI agents with their prompts, manages cycles, and coordinates via shared context files.

**Repo:** https://github.com/ChetanSarda99/OrchyStraw

---

## Tech Stack

- **Orchestrator:** Bash (POSIX-compatible where possible)
- **Scripts:** Bash, Python (for complex parsing/analysis)
- **Config:** Markdown + plain text (agents.conf, prompts)
- **Philosophy:** Zero external dependencies for core. Python OK for tooling.

## Current Tasks

### v0.1.0 — Orchestrator Hardening
1. **Error handling** — graceful failure when agent crashes mid-cycle
2. **Signal trapping** — SIGINT/SIGTERM cleanup (kill child processes, save state)
3. **Cycle state persistence** — save/resume cycle count across restarts
4. **Agent timeout** — configurable per-agent timeout with kill escalation
5. **Dry run mode** — `--dry-run` flag that shows what WOULD run without spawning agents
6. **Logging** — structured cycle logs to `logs/cycle-N.log`
7. **Config validation** — validate agents.conf syntax before running
8. **Lock file** — prevent multiple orchestrators running simultaneously
