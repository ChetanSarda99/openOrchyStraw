## What is Memo?

iOS app for ADHD users — one search bar across everything saved (Telegram, Notion, Instagram, Reddit, voice memos). AI categorization into 8 life areas. Calming, ADHD-friendly. Not a notes app — a personal knowledge capture system.

Swift + SwiftUI (iOS 17+ minimum, iOS 26+ for Apple Intelligence), @Observable (NOT ObservableObject), SwiftData, Supabase Swift SDK, NavigationStack (NOT NavigationView), .task (NOT .onAppear), SF Symbols, AVFoundation, Vision framework, EventKit, LocalAuthentication, StoreKit 2, async/await.

## Backend APIs (Reference)

Base URL per environment. Auth: `Authorization: Bearer <supabase_token>`.
Response: `{ data: T }` / `{ data: T[], meta: { page, limit, total, totalPages } }` / `{ error: { code, message } }`

**New from recent cycles:**
- GET /users/me — includes `trial: { isTrialing, trialEndsAt, trialDaysRemaining }`
- PATCH /users/me/plan — `{ plan: 'free'|'pro', originalTransactionId, productId? }` → updated user
- POST /sync/app-store — Apple server-to-server webhook (uses appAccountToken as userId)
- GET /sources — X-Source-Limit, X-Source-Count headers
- POST /sources/:type/connect — returns 403 SOURCE_LIMIT_EXCEEDED when at free limit (3)
- Cache-Control headers on GET /notes (30s), GET /notes/para (120s), GET /categories (300s)
- X-Quota-* headers on POST /notes, /actions, /checklists, /autofill
- X-RateLimit-* headers on all protected routes (120 req/min per user)
- 502 on Claude/Voyage failures (ExternalServiceError)
