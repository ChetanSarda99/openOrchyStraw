## v0.1.0

Initial release — orchestrator hardening.

### Core orchestrator
- 8 bash modules: logger, error-handler, cycle-state, agent-timeout, dry-run, config-validator, lock-file, bash-version
- All modules double-source guarded
- Portable shebang (`#!/usr/bin/env bash`)
- Bash 5.0+ version check

### Safety
- File ownership enforcement with rogue write detection
- Protected file restoration
- Prompt validation (restore from backup if corrupted)
- Usage limit pausing (>=70% threshold)
- Empty cycle detection (auto-stop after 3)

### Testing
- 12 unit test files + integration test
- Test runner script
- All tests passing
