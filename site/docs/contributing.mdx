## Getting started

OrchyStraw is open source under the MIT license. Contributions are welcome.

```bash
# Fork and clone
git clone https://github.com/YOUR_USERNAME/openOrchyStraw.git
cd openOrchyStraw

# Make the orchestrator executable
chmod +x scripts/auto-agent.sh
```

## Guidelines

- **Keep it simple** — OrchyStraw's strength is zero dependencies. Don't add package managers, build tools, or runtimes.
- **Bash only** — Core orchestrator code must be bash. No Python, no Node.js, no compiled languages.
- **Test your changes** — Run `./scripts/auto-agent.sh --dry-run` to verify config changes work.
- **One PR per change** — Small, focused PRs are reviewed faster than large bundles.

## Reporting issues

Open issues on the [openOrchyStraw GitHub repo](https://github.com/ChetanSarda99/openOrchyStraw/issues). Include:

1. OrchyStraw version (or commit hash)
2. Bash version (`bash --version`)
3. OS and AI tool being used
4. Steps to reproduce
5. Error output or logs
