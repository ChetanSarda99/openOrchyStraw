# Changelog

## v0.1.0 — March 16, 2026

First public release.

### What's included
- **Orchestrator script** (`auto-agent.sh`) — runs full dev cycles with git branching, file ownership enforcement, rogue write detection, prompt backups
- **Template folder** — copy into any project to get started
- **Bootstrap prompt** — auto-generates agents based on your codebase
- **Agent design guide** — prompt structure, skills layers, common mistakes
- **Usage guides** — Claude Code, Windsurf, Cursor, Codex, Aider, Gemini
- **Concepts doc** — detailed explainer of every component
- **Custom agent guide** — how to add, configure, and design new agents
- **Examples** — sample `agents.conf` and PM prompt
